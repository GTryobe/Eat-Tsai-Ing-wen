## 关于蔡英文

[蔡英文-百度百科](https://baike.baidu.com/item/蔡英文/353)2016年5月20号，蔡英文当选台湾“省长”，2020年1月11号连任，多次以“民主”的名号搞台独，她从未承认“九二共识”，企图将台湾省从中华人民共和国分离出去，为了实现自己的野心，不惜牺牲台湾省人民。她纵容民进党民代使用暴力，支持“反服贸抗争”，对是否兼任党首持双重标准，无恶不赦。
